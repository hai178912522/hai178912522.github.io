7.1. 索引

到目前为止，我们已经从数据集中删除了低质量细胞、环境RNA污染和双倍体，数据以计数矩阵的形式作为形状为细胞数×基因数的数字矩阵可用。这些计数表示单细胞RNA测序实验中分子的捕获、反转录和测序。这些步骤中的每一步都会为相同的细胞测量计数深度添加一定的变异度，因此计数数据中细胞间基因表达的差异可能仅仅是由于采样效应造成的。这意味着数据集和因此计数矩阵仍然包含广泛变化的方差项。分析数据集通常是具有挑战性的，因为许多统计方法都假设具有均匀方差结构的数据。

伽马-泊松分布
UMI数据的理论和实证建立的模型是伽马泊松分布，它意味着平方均方差关系，$\operatorname{Var}[Y]=\mu+\alpha \mu^{2}$
，平均值为 $μ$和过度离群为$α$。对于$α=0$，这是Poisson分布，描述了超过Poisson分布的额外方差。”

“归一化”数据预处理步骤的目的是通过将可观测方差缩放到指定范围，对数据集中的原始计数进行调整以处理变量采样效应。实践中使用多种不同的归一化技术，复杂程度也各不相同。它们大多设计成这样的方式，以便后续分析任务和它们的基础统计方法可以应用。

Ahlmann-Eltze和Huber在2022年发表的最近一项基准测试[Ahlmann-Eltze and Huber, 2022]比较了22种单细胞数据转换方法的效果。该基准测试基于细胞图形与基本事实的重叠情况比较了不同归一化技术的表现。我们想强调的是，仍未有一份完整的基准测试，该测试还比较了归一化对各种不同的下游分析任务的影响。我们建议分析师仔细选择归一化方法，并始终依赖于后续分析任务。

本章将向读者介绍三种不同的归一化技术，即移位对数转换、scran归一化和Pearson残差的分析逼近。移位对数有助于稳定后续的维度缩减和鉴定差异表达基因。Scran已经过广泛测试，并用于批量校正任务，分析Pearson残差尤其适用于选择具有生物变异性的基因和识别稀有细胞类型。

7.2. 移位对数#
我们将介绍的第一种规范化技术是移位对数，它基于三角法 [Dorfman，1938]。 三角法将非线性函数
$f(Y)$ 应用于原始计数 $Y$并旨在使数据集中的方差更加相似。
通过移位对数来解决这个问题。
$f(y)=\log \left(\frac{y}{s}+y_{0}\right)$
以$y$为原始计数，以$s$为所谓的大小因子，以$y0$描述假计数。大小因子为每个细胞确定，以考虑采样效应和不同细胞大小的变化。
细胞$c$的大小因子可以通过计算得出。
$s_{c}=\frac{\sum_{c} y_{g c}}{L}$

对于一个基因$g$和$L$描述目标总和的基因组，有不同的方法来确定数据中的大小因子。我们将在本节中利用Scanpy的默认值，其中$L$是数据集中原始计数深度的中位数。许多分析模板使用固定值$L$，例如$L=10^{5}, \text { or } L=10^{6}$，导致通常称为每百万计数（CPM）的值。对于初学者来说，这些值可能看似任意，但它可以导致比单细胞数据集中通常看到的更大的过度分散。

过度离散（Overdispersion）
过度离散度是指数据集中存在比预期更大的变异性。

“移位对数”是一种快速归一化技术，优于其他方法，可揭示数据集的潜在结构（特别是在主成分分析之后）并有助于稳定方差，以进行后续降维和识别差异表达基因。现在我们将检查如何将此归一化方法应用于我们的数据集。通过在target_sum = None的情况下运行pp.normalized_total，可以方便地使用scanpy调用移位对数。由于我们希望在本教程中探索三种不同的归一化技术，因此将inplace参数设置为False。现在，第二步使用缩放计数，并且我们获得了第一次归一化计数矩阵。

这段代码的作用是对adata数据进行归一化和log1p转换处理。
首先，sc.pp.normalize_total()函数将adata中的每个细胞上的每个基因在所有细胞共现的基因数量的比例进行归一化，可以指定目标和也可以不指定，然后返回一个经过归一化的大小比例数据。这个数据被赋值给了scales_counts。
接下来，sc.pp.log1p()函数对scales_counts[“X”]进行单调递增的log1p转换，这个转换可以压缩值域，将数值偏高的基因表达值进行压缩，使得整个数据符合更好的正态分布。结果被保存在adata.layers[“log1p_norm”]中。 若要得到这变换后的数据，可以通过adata[:, :] 调用。
如果inplace=True，则可以将这些变换应用到原始数据上，而不是像这样在副本上进行。

第二种基于delta方法的归一化方法是Scran的基于池化的大小因子估计方法。Scran遵循与移位对数相同的原则，通过计算$f(y)=\log \left(\frac{y}{s}+y_{0}\right)$描述一个伪计数。现在唯一的区别是，Scran利用反卷积方法，基于细胞池的基因进行线性回归来估计大小因子。该方法旨在更好地考虑数据集中所有细胞的计数深度差异。

细胞被分割成池，并且Scran使用基于基因的线性回归估计池基尺度因子。Scran已经广泛用于批次校正任务，并且可以很容易地使用相应的R包来调用。

scran 需要一个粗糙聚类输入来提高规模因子估计性能。在本教程中，我们使用简单的预处理方法，并以低分辨率对数据进行聚类，以获得输入以进行规模因子估算。基本预处理包括假设所有规模因子相等（库大小归一化为百万计数 - CPM），并对计数数据进行对数变换。

这段代码是用于数据分析中的基本聚类过程。它对数据进行了不同的归一化处理，并使用PCA方法对数据进行降维处理。随后，使用邻域图构建近邻图，并使用Leiden算法进行聚类，以便将数据划分为不同的组。
具体来说，代码的运行过程如下：

  • adata_pp = adata.copy()原始数据被复制到新变量adata_pp中。
  • sc.pp.normalize_total(adata_pp)使用normalize_total函数对数据进行总体归一化，以便在后续的处理中将不同的细胞进行比较。
  • sc.pp.log1p(adata_pp)使用log1p函数对数据进行对数变换，以便使数据的分布更加接近正态分布。
  • sc.pp.pca(adata_pp, n_comps=15)使用PCA方法对数据进行降维处理，并将n_comps参数设置为15，以便保留15个主成分。
  • sc.pp.neighbors(adata_pp)使用邻域图构建近邻图，以便在下一步聚类时考虑细胞之间的相似度。
  • sc.tl.leiden(adata_pp, key_added="groups")使用Leiden算法对数据进行聚类，并将聚类结果存储在key_added参数所指定的变量中。

通过这些预处理步骤，原始数据可以更好地适应后续的聚类算法，从而在分类和识别细胞类型时发挥更好的作用。

我们现在将 data_mat 和我们计算的组添加到我们的 R 环境中。

  1. 将adata_pp对象的数据矩阵转置，得到转置后的矩阵data_mat；
  2. 判断data_mat是否为稀疏矩阵，若是稀疏矩阵，则进行转换，并将其转换为COO或CSC格式；
  3. 将Python环境中的data_mat矩阵通过R语言的robjects包转换为R语言中的全局变量data_mat；
  4. 将adata_pp对象中的obs数据框的“groups”列赋值给R语言中的全局变量input_groups。

现在我们也可以删除 anndata 对象的副本了，因为我们已经获得了运行 scran 所需的所有对象。

我们现在根据之前计算的细胞组计算大小因子。

这段代码是利用R语言进行转录组数据的标准化处理，特别是single-cell RNA-seq（单细胞RNA序列）的数据处理。主要功能是计算大小因子（size factors），用于调整每个样本的reads（reads数）数量，从而保证不同样本之间的reads计数可以进行比较。下面是代码的解释：

  • “%%R -o size_factors” 表示在jupyter notebook（一个交互式笔记本）中运行R语言并输出size_factors的结果。
  • “size_factors = sizeFactors()” 定义了一个名为size_factors的R对象，它将存储所有计算结果。
  • “computeSumFactors()” 函数计算每个样本的规格化因子（normalization factors），它们根据每个样本中基因的总读数以及每个基因的可变性来确定。
  • “SingleCellExperiment()” 函数以数据框（matrix）的形式载入数据。
  • “clusters = input_groups” 表示将数据分组，用于计算每个样本的规格化因子。
  • “min.mean = 0.1” 表示过滤掉低reads数量的样本。
  • “BPPARM = Multicore Param()” 是R语言实现并行计算的一种方式，以加快计算速度。

我们在.obs文件中保存了size_factors，现在可以对数据进行归一化处理，随后应用log1p变换。

第一步是为adata添加一个名为”size_factors”的观测值，值为变量”size_factors”。第二步是进行一个简单的数学计算，将adata矩阵X中的每个元素除以其对应观测值中的”size_factors”值。第三步是在adata中添加一层，并将处理后的矩阵储存在其中，该层名为”scran_normalization”。在处理后的矩阵中，使用sc.pp.log1p对每个元素取对数并加上1，生成一个稀疏矩阵。

7.3. 分析皮尔逊残差

在本章中，我们介绍的第三种归一化技术是Pearson残差的分析逼近法。这种归一化技术的动机是观察到scRNA-seq数据中细胞间变异性可能会受到生物学异质性和技术效应的干扰。该方法利用“负二项式回归的正则化Pearson残差”来计算数据中的技术噪声模型。它在广义线性模型中明确地增加了计数深度作为共变量。[Germain等人，2020]在不同归一化技术的独立比较中表明，该方法消除了采样效应的影响，同时保留了数据集中的细胞异质性。值得注意的是，分析Pearson残差不需要像伪计数增加或对数转换这样的下游启发式步骤。该方法的输出是可以是正值也可以是负值的归一化值。对于细胞和基因而言，负的残差表示观察到的计数比基因的平均表达和细胞测序深度所预期的要少。正残差表示计数更多。分析Pearon残差已经在scanpy中实现，并且可以直接在原始计数矩阵上计算。

这段代码用于对adata中的基因表达矩阵进行归一化处理，具体步骤如下：

  1. 使用Scanpy中的sc.experimental.pp.normalize_pearson_residuals函数计算pearson残差，并进行归一化处理，返回值为一个Anndata对象，其中包含了处理后的归一化pearson残差矩阵。
  2. 将处理后的pearson残差矩阵存储在adata的layers中，命名为”analytic_pearson_residuals”。

其中，参数inplace=False表示不改变原始的adata对象，而是返回一个新的Anndata对象。csr_matrix函数则将矩阵转化为压缩稀疏行格式，以便在内存占用上更加高效。


我们对数据集应用了不同的归一化技术，并将它们保存为独立的层到我们的anndata对象中。根据下游分析任务的需要，使用不同归一化的层并评估结果可能更有利。
